# Die Kampfumgebung

## Das Layout 
- Raumbild im  Hintergrund
- unten links der Schüler
- unten rechts der Lehrer
- Info Boxen über beiden
- eine große Box mit Items und "Aufgeben" Button ganz oben

## Ablauf

### Begegnung

- Schüler trifft auf Lehrer 
- -> klickt auf Kampf Button
- Übergangsszene  
  - schwarzes -> wieder hell
  - „Frohes Schaffen …“  
- Kampf findet in Gang der Fachschaft statt
- Lehrer startet

---

### Lehrerzug

- Benutzt Item seiner Fachschaft  
- Verteidigt
- kleine Animation?

---

### Schülerzug

- Auswählen von Item innerhalb von 10 s/ Aufgeben
- kleine Animation?

---

### Ende

- Aufgeben → Lehrer gewinnt
- a-b Schaden
- Schüler hat keine HP mehr → Spruch von Lehrer  
- Lehrer keine HP mehr → erhält Item von Lehrer
