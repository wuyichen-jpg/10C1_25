\begin{frame}
	\frametitle{Items}
	Was sind Items?
	\begin{itemize}
		\item Objekte, die zum Kampf benötigt werden
		\item Erhalten eines Starter-Items zu Beginn des Spieles, nachdem man sich in eine der fünf Fachschaften eingewählt hat
		\item Erhalten weiterer Items nach erfolgreichen Abschluss von Minigames
		\item Speicherung und Zugriff auf alle Items im persönlichen Inventar
	\end{itemize}
\end{frame}
